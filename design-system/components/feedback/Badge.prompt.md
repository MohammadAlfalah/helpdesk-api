Generic labelled pill — categories, counts, arbitrary tags.

```jsx
<Badge tone="brand">Network</Badge>
<Badge tone="neutral" dot>12 open</Badge>
<Badge tone="success" icon="check">Done</Badge>
```

Tones: `neutral` / `brand` / `info` / `warning` / `success` / `danger`. For ticket **status** and **priority**, use the domain components `StatusBadge` / `PriorityBadge` instead — they carry the canonical color mapping.
